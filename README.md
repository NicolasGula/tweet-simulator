## Instalacion

1. Clona el repositorio
   ```sh
   git clone https://github.com/NicolasGula/tweet-simulator.git
   ```
2. Instala las dependencias con YARN
   ```sh
   yarn install
   ```
3. Corre la aplicacion en modo dev
   ```js
   yarn dev
   ```
